# Executive Secretary V37

Ready-to-deploy Next.js + Tailwind project for Vercel.

## Local run

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Go to Vercel.
3. Click **Add New Project**.
4. Select the GitHub repository.
5. Framework should be detected as **Next.js**.
6. Click **Deploy**.
